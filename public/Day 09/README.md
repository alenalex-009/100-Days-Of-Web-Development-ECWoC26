# Day 9 – QuickNotes Pro

## 📌 Overview
QuickNotes Pro is an enhanced note-taking application built using HTML, CSS, and JavaScript.

It includes advanced UI styling and multiple interactive features.

---

## 🚀 Features

- Add, Edit, Delete Notes
- Search Functionality
- Note Color Selection
- Category Selection
- Date & Time Stamp
- Drag & Drop Reordering
- Smooth Animations
- Dark / Light Mode
- Full localStorage Persistence
- Responsive Design

---

## 💾 LocalStorage

All notes are stored in browser localStorage:
- Notes persist after refresh
- Order is preserved
- Updates are saved automatically

---

## 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- localStorage API
