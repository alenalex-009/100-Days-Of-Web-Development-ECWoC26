# 📊 Social Media Dashboard

A modern **Social Media Analytics Dashboard** that simulates activity across multiple platforms like **Facebook, Twitter, Instagram, and YouTube**.

This dashboard provides a visual overview of engagement metrics such as followers, likes, comments, shares, and views, along with interactive features like post creation, story simulation, and engagement tracking.

---

## 🚀 Features

### 📈 Platform Overview

Displays follower/subscriber counts for major social media platforms:

* Facebook Followers
* Twitter Followers
* Instagram Followers
* YouTube Subscribers

Animated counters simulate follower growth to create a realistic dashboard experience.

---

### ✍️ Post Creation

Users can simulate creating posts directly from the dashboard.

Features:

* Write post content
* Select platform (Facebook, Twitter, Instagram, YouTube)
* Publish post to the activity feed

Posts appear instantly in the **Recent Posts section**.

---

### 📱 Story Simulation

The dashboard includes a **story feature** similar to social media platforms.

Features:

* Add a new story
* Stories appear as preview cards
* Stories automatically disappear after a short duration

---

### ❤️ Interactive Engagement

Posts include interactive buttons:

* 👍 Like
* 💬 Comment
* 🔁 Share

When users interact with posts, the **dashboard metrics update dynamically**.

---

### 📊 Engagement Analytics

The dashboard uses **Chart.js** to display engagement statistics including:

* Likes
* Comments
* Shares
* Views

The chart updates automatically as users interact with posts.

---

### 📋 Activity Feed

The **Recent Posts section** shows posts published by the user.

Each post includes engagement buttons to simulate real social media activity.

---

### 🌓 Dark Mode

Users can toggle between **Light Mode and Dark Mode**.

The selected theme is saved using **localStorage**, so it persists after page refresh.

---

### ⏱ Live Date and Time

The dashboard displays the **current date and time** and updates every second.

---

### 🔔 Welcome Notification

A toast notification appears when the dashboard loads to simulate real platform alerts.

---

## 🛠️ Technologies Used

* **HTML5** – Structure and layout
* **CSS3** – Styling, responsive design, animations
* **JavaScript (Vanilla JS)** – Interactivity and dynamic updates
* **Chart.js** – Data visualization
* **LocalStorage API** – Theme persistence
