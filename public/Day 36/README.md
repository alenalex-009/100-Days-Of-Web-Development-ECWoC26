# 🏋️ Fitness Tracker Dashboard

A modern **Fitness Tracker Dashboard** that allows users to log their workouts, track calories burned, monitor workout duration, and visualize their fitness progress through charts and statistics.

This project focuses on improving the **UI, dashboard experience, and activity tracking features** to make the application more interactive and user-friendly.

---

## 🚀 Features

### 📊 Dashboard Statistics

The dashboard displays key fitness metrics in real-time:

* **Total Activities Logged**
* **Total Workout Duration**
* **Total Calories Burned**

These statistics update dynamically whenever a new activity is added.

---

### 📝 Add Activity

Users can log their workouts by entering:

* Activity Name (Running, Yoga, Gym, etc.)
* Duration in minutes
* Calories burned

All activities are instantly stored and displayed on the dashboard.

---

### 📈 Fitness Progress Visualization

The application uses **Chart.js** to display a visual representation of calories burned per activity.

This helps users easily understand their fitness progress.

---

### 🎯 Daily Goal Tracking

The dashboard includes a **daily calorie goal progress bar**.

* Default Goal: **500 kcal**
* Progress updates automatically based on logged activities.

---

### 📜 Activity History

All activities are shown in a **clean activity history section**, including:

Example:

```
Running — 30 min — 250 kcal  
Yoga — 20 min — 120 kcal
```

This helps users review their past workouts.

---

### 💡 Smart Workout Recommendation

Based on total calories burned, the dashboard provides simple recommendations such as:

* Try walking or stretching
* Increase activity level
* Great progress, keep going
* Goal achieved

---

### 💾 Local Storage Support

All activities are saved using **localStorage**, so the data remains available even after refreshing the page.

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Modern UI styling and responsive layout
* **JavaScript (Vanilla JS)** – Application logic and activity tracking
* **Chart.js** – Data visualization and charts
* **LocalStorage API** – Data persistence
