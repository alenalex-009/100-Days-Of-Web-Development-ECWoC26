# 🌍 GlobalX – Currency Converter Dashboard

## 📌 Overview

GlobalX is a modern SaaS-style Currency Converter Dashboard built using HTML, CSS, and JavaScript.

It allows users to convert currencies between multiple countries using predefined exchange rates and provides a clean, professional UI with analytics visualization.

---

## ✨ Features

- ✅ Professional SaaS Dashboard UI
- ✅ Sidebar Navigation (Dashboard, Countries, Analytics)
- ✅ Bidirectional Currency Conversion (e.g., INR ↔ USD)
- ✅ Swap Button Functionality
- ✅ Reverse Exchange Rate Display
- ✅ Supported Countries Section with Flag Images
- ✅ Analytics Section with Exchange Rate Chart
- ✅ Responsive Design
- ✅ Clean Dark Theme UI
- ✅ Smooth Hover Effects

---

## 🌎 Supported Currencies

- USD (United States Dollar)
- INR (Indian Rupee)
- GBP (British Pound)
- EUR (Euro)
- JPY (Japanese Yen)
- CAD (Canadian Dollar)

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (Vanilla JS)
- Chart.js (Analytics Visualization)

## 🔄 How Conversion Works

1. Select a **From Country**
2. Select a **To Country**
3. Enter the amount
4. Click **Convert**

The system:
- Converts the value using exchange rate logic
- Displays the converted result
- Shows reverse rate for better clarity

Example:
100 INR = 1.20 USD
1 USD = 83.0000 INR
