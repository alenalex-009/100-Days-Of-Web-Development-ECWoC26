# 🎲 Modern Dice Roller

A modern **Dice Roller Web Application** built with **HTML, CSS, and JavaScript**.
This project simulates rolling a dice with smooth animations, modern UI styling, and interactive controls.

The application provides a visually appealing experience with animated dice rolling, responsive design, and theme switching.

---

## 🚀 Features

### 🎨 Modern UI Design

The dice roller includes a clean and modern interface featuring:

* Glassmorphism style card layout
* Smooth gradients and animated background
* Modern typography and spacing
* Responsive design for multiple screen sizes

---

### 🎲 Dice Roll Simulation

The application simulates a real dice roll.

Features include:

* Random dice number generation (1–6)
* Smooth dice rolling animation
* Visual feedback after rolling
* Result display for each roll

---

### 🔄 Reset Functionality

Users can reset the game at any time.

Reset features:

* Reset dice state
* Reset roll counter
* Clear current result display

---

### 📊 Roll Counter

The application tracks how many times the dice has been rolled.

Example display:

```text
Total Rolls: 5
```

This helps users monitor their interactions with the dice.

---

### 🌗 Dark / Light Mode

The dice roller includes a **theme toggle** that allows switching between:

* Dark Mode
* Light Mode

This improves accessibility and user experience.

---

### ✨ Smooth Animations

The project includes several animations to enhance interaction:

* Dice rolling animation
* Button hover effects
* Background gradient animation
* Smooth transitions between UI states

---

## 🛠 Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling, layout, animations
* **JavaScript (Vanilla JS)** – Dice logic and interactivity
