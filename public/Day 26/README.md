# 📸 Image Gallery

A modern **Image Gallery Web Application** that allows users to browse, search, filter, and view images in an interactive and responsive layout.

This project enhances a basic gallery by introducing **image categories, filtering, captions, search functionality, and a lightbox viewer**, making it closer to a real-world media gallery.

---

## 🚀 Features

### 🖼 Image Grid Layout

Images are displayed in a **responsive grid layout** that automatically adjusts for different screen sizes.

* Desktop → Multiple columns
* Tablet → Medium grid layout
* Mobile → Single-column view

---

### 🗂 Category-Based Gallery

Images are organized into categories for better structure.

Available categories include:

* 🌿 Nature
* 🐶 Animals
* 🍔 Food
* ✈️ Travel

Users can easily switch between categories using **filter buttons**.

---

### 🔍 Image Search

Users can search images using the **search bar**.

The gallery dynamically filters images based on the entered text (caption search).

---

### ➕ Add New Images

Users can add new images directly to the gallery.

Required fields:

* Image URL
* Caption (optional)
* Category

New images are saved using **localStorage**, so they remain available after refreshing the page.

---

### 🖱 Hover Effects

Images include interactive hover effects such as:

* Smooth zoom animation
* Caption overlay
* Modern card design

These effects improve the visual appearance and user experience.

---

### 🔍 Lightbox Viewer

Clicking an image opens it in a **fullscreen lightbox modal**.

Features:

* Large image preview
* Caption display
* Close button
* Escape key support

---

### 💾 Local Storage Support

All added images are saved using **LocalStorage**, ensuring that user-added images persist across page reloads.

---

## 🛠 Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling, layout, animations
* **JavaScript (Vanilla JS)** – Interactivity and dynamic filtering
* **LocalStorage API** – Data persistence
