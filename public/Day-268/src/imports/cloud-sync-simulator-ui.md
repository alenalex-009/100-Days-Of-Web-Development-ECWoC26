Design a modern web-based simulation dashboard UI titled “Cloud Sync Simulator.” The application should visually demonstrate how files sync between local devices and cloud storage systems. It should show upload/download progress, sync conflicts, version history, and network behavior in real time. The interface should feel similar to tools like Google Drive sync, Dropbox, or OneDrive dashboards, with a clean and professional cloud-inspired aesthetic.
